# discpp

Just a small pet library written in C++ intended to power Discord bots.

## Dependencies

The library currently requires C++14, and the following third-party dependencies
(as well as their transitive dependencies).

- nlohmann/json
- boost::beast >=1.68
- boost::log
- cmake, make
